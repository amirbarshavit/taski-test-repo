# Taski

Taski is a lightweight task management application that helps teams track, prioritize, and complete work items efficiently. It streamlines project workflows by providing a simple interface for creating and managing tasks across multiple projects.

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/amirbarshavit/taski.git
cd taski
pip install -r requirements.txt
```

## Usage

Run the application with:

```bash
python main.py
```

## Configuration

Taski can be configured via environment variables or a `.env` file in the project root. Refer to `.env.example` for available options.

### Business Overview

SentinelOne provides an AI-driven cybersecurity platform called **Singularity**, which offers:

- **Endpoint Protection Platform (EPP)** – Prevents malware, ransomware, and zero-day exploits.
- **Endpoint Detection and Response (EDR)** – Detects and responds to threats in real time.
- **Extended Detection and Response (XDR)** – Unifies security data across the enterprise.
- **Cloud Security** – Protects cloud workloads, containers, and Kubernetes environments.
- **Identity Security** – Secures Active Directory and hybrid identity environments.
- **Data Lake (Singularity Data Lake)** – Ingests and correlates petabytes of security data.

#### Bulls (Why Buy S1?)
- **Massive TAM**: The global cybersecurity market is projected to exceed $500B by 2030.
- **AI-Native Platform**: SentinelOne is built on AI from the ground up, unlike legacy competitors.
- **Strong ARR Growth**: Consistent double-digit ARR growth signals strong product-market fit.
- **Land & Expand Model**: Customers regularly expand their usage, boosting net revenue retention (NRR).
